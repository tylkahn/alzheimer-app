# alzheimer-app


# dev notes
- requirements: python 3.7+, pip, pipenv
- always run `pipenv shell` before doing any backend stuff
- backend must be started before frontend, or the frontend will complain
## initial setup
- `cd backend`
- `pipenv shell`
- `pipenv install`

-----------------

- `cd frontend`
- `npm install`

## start backend
- `cd backend`
- `pipenv shell`
- `python manage.py runserver`

## start frontend
- `cd frontend`
- `npm start`
